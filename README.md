This project aims to develop a web-based worker tracking application for hazardous environments.
The system allows safety supervisors to monitor the location and activity status of workers operating in dangerous areas such as industrial plants, construction sites, or disaster zones.

The application provides a centralized dashboard with map-based localization, worker status monitoring, and emergency alerts when a worker becomes unresponsive or triggers a distress signal.
The solution is intended as a prototype, using simulated data to demonstrate functionality and support worker safety management.
